# aiu
`aiu` stands for `AI for You`

##
AI - LLM, RAG, Agentic

## llm
llm dir has LLM related files - working code

## rag
rag dir has RAG related files - working code

## a2a (agent2agent | agent-to-agent)
a2a dir has Agent-to-Agent related files - working code

## models
Popular models you can pull from the registry include:
* General Purpose: `llama3.2`, `gemma3`, `mistral`
* Coding: `qwen3-coder`, `devstral`
* Reasoning: `deepseek-r1`, `nemotron-3-super`
